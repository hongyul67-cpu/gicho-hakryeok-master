# 📚 기초학력 마스터 — 국·영·수 학습·진단

중학생 대상 국어·영어·수학 기초학력 학습 및 진단 도구입니다. 60문항을 풀면 자동 채점되고,
결과를 이미지로 저장해 남길 수 있습니다.

🔗 **바로가기:** https://hongyul67-cpu.github.io/gicho-hakryeok-master/

## 주요 기능
- 국·영·수 **60문항 진단** · 자동 채점
- **결과 이미지 출력** — 진단 결과를 이미지로 저장
- 영역별 성취 확인

## 결과수집
학생의 진단 결과를 교사 구글시트로 제출할 수 있습니다. (공유 링크의 `?rc=` 파라미터로 활성화)

## 사용 방법
- 위 주소를 열면 바로 사용할 수 있습니다. 설치가 필요 없습니다.
